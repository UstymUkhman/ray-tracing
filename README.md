## Download ##

`git clone https://github.com/UstymUkhman/ray-tracing.git`

`cd ray-tracing`

## Develop ##

`pnpm i`

`pnpm start`

## Lint ##

`pnpm lint:js`

`pnpm lint:css`

## Build ##

`pnpm build`

`pnpm serve`
